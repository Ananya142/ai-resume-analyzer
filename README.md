
# AI Resume Analyzer using NLP

An AI-powered resume screening system that analyzes resumes and job descriptions using Natural Language Processing (NLP) to calculate match scores, identify skill gaps, and provide explainable candidate insights through an interactive dashboard.


##  Features

* Resume parsing and text extraction
* Job description analysis
* Skill matching using NLP
* Match score calculation
* Skill gap identification
* Explainable candidate insights
* Interactive dashboard for visualization


## Tech Stack

**Frontend**

* React
* TypeScript
* HTML5
* CSS3


**NLP / AI**

* NLP techniques (TF-IDF / cosine similarity / keyword extraction)
* spaCy / NLTK (if used)

**Database**

* MySQL / SQLite (if applicable)
